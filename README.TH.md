# Camera Adjustment Simulation

Camera Adjustment Simulation เป็นแอปสอนการถ่ายภาพแบบอินเทอร์แอคทีฟสำหรับสำรวจว่าการปรับค่ากล้องส่งผลต่อภาพอย่างไร แอปนี้ใช้ SVG, CSS และ TypeScript เพื่อแสดงความสัมพันธ์ระหว่างความยาวโฟกัส, รูรับแสง, ระยะโฟกัส, ระยะของแบบ, ระยะฉากหลัง, ขนาดเซนเซอร์, เปอร์สเปกทีฟ, โบเก้, การเปิดรับแสง และความบิดเบี้ยวของเลนส์

โปรเจกต์นี้ออกแบบให้เป็นเครื่องมือเรียนรู้ขนาดกะทัดรัด ไม่ใช่ซิมูเลเตอร์กล้องระดับโปรดักชัน แผนภาพต่าง ๆ ให้ความสำคัญกับการอธิบายด้วยภาพที่เข้าใจง่ายและการโต้ตอบที่สอดคล้องกัน มากกว่าการจำลองออปติกอย่างละเอียดครบถ้วนทุกกรณี

## แอปนี้ทำอะไร

แอปนี้มีหน้าอินเทอร์แอคทีฟ 8 หน้า:

- **Exposure Triangle**: แสดงว่าค่า f-stop, ความเร็วชัตเตอร์ และ ISO ส่งผลต่อความสว่าง, ภาพเบลอจากการเคลื่อนไหว และนอยส์อย่างไร
- **Perspective Compression**: แสดงว่าระยะกล้อง, ระยะแบบ, ระยะฉากหลัง และความยาวโฟกัส ส่งผลต่อขนาดที่มองเห็นของฉากหลังอย่างไร
- **Depth Diagram**: แสดงแผนภาพออปติกมุมมองจากด้านบน พร้อมตัวกล้อง, เลนส์, ระนาบเซนเซอร์/ฟิล์ม, แบบ, ฉากหลัง, มุมรับภาพ, ระยะโฟกัส, ขอบเขตชัดใกล้, ขอบเขตชัดไกล, ระยะไฮเปอร์โฟกัส และระยะชัดลึกจริง
- **Liveview Simulation**: แสดงฉากจำลองแบบช่องมองภาพ/ไลฟ์วิวอย่างง่าย มีนางแบบและฉากหลังเมือง โดยใช้ค่ากล้องชุดเดียวกับแผนภาพระยะชัดลึก
- **Sensor Crop Comparison**: เปรียบเทียบกรอบภาพของฟูลเฟรม, APS-C และ Micro Four Thirds เมื่อใช้ความยาวโฟกัสเดียวกัน
- **Bokeh Shape Simulator**: แสดงรูปทรงช่องรูรับแสงจากใบเบลดอย่างง่าย และพฤติกรรมของไฮไลต์ฉากหลังที่อยู่นอกโฟกัส
- **Focus Plane / Zone Focus**: แสดงเครื่องหมายระยะชัดใกล้, ระยะโฟกัส และระยะชัดไกลบนสเกลระยะทาง
- **Lens Distortion**: แสดงกริดอ้างอิงและภาพเปรียบเทียบบุคคลอย่างง่าย สำหรับพฤติกรรมความบิดเบี้ยวแบบเลนส์มุมกว้าง, เลนส์มาตรฐาน และเลนส์เทเลโฟโต

## วัตถุประสงค์

ซอฟต์แวร์นี้มีไว้เพื่อช่วยให้ช่างภาพ, นักเรียน, ผู้สอน และผู้เรียนรู้ด้วยภาพ เข้าใจแนวคิดของกล้องที่มักเข้าใจยากเมื่อดูจากตัวเลขเพียงอย่างเดียว

เป้าหมายหลักคือ:

- ทำให้ความสัมพันธ์ของค่ากล้องมองเห็นได้และปรับโต้ตอบได้
- เชื่อมคำศัพท์ทางเทคนิคของกล้องเข้ากับพฤติกรรมที่แสดงในแผนภาพ
- รองรับศัพท์การถ่ายภาพทั้งภาษาอังกฤษและภาษาไทย
- ให้เป็นสื่อการสอนบนเบราว์เซอร์ที่สะอาดและรันในเครื่องได้

## ค่าเริ่มต้นปัจจุบัน

ค่ากล้องเริ่มต้นคือ:

- ความยาวโฟกัส: `50 mm`
- F-stop: `f/4.0`
- ระยะโฟกัส: `5.0 m`
- ระยะแบบ: `5.0 m`
- ระยะฉากหลัง: `15.0 m`

พรีเซ็ตเซนเซอร์ที่รองรับ:

- Full frame
- APS-C
- Micro Four Thirds

## การรันในเครื่อง

ติดตั้ง dependency:

```bash
npm install
```

เริ่มเซิร์ฟเวอร์สำหรับพัฒนาในเครื่อง:

```bash
npm run dev
```

สร้างเวอร์ชัน production:

```bash
npm run build
```

รันชุดตรวจสอบมาตรฐาน:

```bash
npm run verify
```

รันชุดตรวจสอบเต็ม รวมถึง Playwright tests:

```bash
npm run verify:full
```

## โครงสร้างโปรเจกต์

- `src/core/`: ค่าคงที่ของกล้อง, การคำนวณออปติก, helper สำหรับจัดรูปแบบข้อความ และฟังก์ชันวัดค่าของแต่ละหน้า
- `src/pages/`: renderer ของ SVG และภาพสำหรับแต่ละหน้า
- `src/i18n.ts`: ป้ายข้อความภาษาอังกฤษและภาษาไทย, ชื่อหน้า, สรุปหน้า และศัพท์เทคนิคการถ่ายภาพ
- `tests/e2e/`: Playwright smoke tests, visual baseline tests, language tests และ accessibility tests
- `PLAN*.md`: แผนออกแบบและแผน implementation ของแต่ละหน้า
- `QUALITY_REPORT.md`: เมทริกซ์คุณภาพโค้ดปัจจุบันและสถานะการตรวจสอบ
- `LICENSE`: license แบบกำหนดเองสำหรับการใช้งาน source-available แบบไม่ใช่เชิงพาณิชย์
